Cấu Hình Supabase Storage

Bảo mật: Tuyệt đối không điền service_role key của Supabase vào đây. Khi đưa file này lên mạng, mọi người đều có thể đọc được source code của bạn. Hãy chỉ sử dụng public anon key đã phân quyền hợp lý!

Hướng dẫn thiết lập Supabase:

  1. Truy cập vào Dashboard Supabase -> Chọn Project của bạn.
  2. Vào tab Storage -> Tạo bucket mới tên là blog-images. Bật tùy chọn Public.
  3. Vào tab Policies của bucket blog-images -> Thêm policy cho phép INSERT cho đối tượng Anonymous/Public (hoặc tất cả mọi người).
